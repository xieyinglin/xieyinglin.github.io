Mardown 基本语法

发表于 2018-08-25 | 分类于 Markdown |

不同的 Markdown 处理器会有些变体和差异，markdown 文件同样可以使用 html 标签。

标题

使用 # 在单词或词组前面(需要在行开头才能生效)，#的个数代表标题的级别，一个#代表一级标题，2个代表2级标题，一共支持6级标题

例如：

#heading1
##heading2
###heading3
####heading4
#####heading5
######heading6

效果：

heading1

heading2

heading3

heading4

heading5
heading6

或者在一行文本下方添加任意数量的 = 代表一级标题，或任意数量的 - 代表二级标题

例如：

一级标题
\=\=\=\=\=\=\=\=

二级标题
--------

效果：

一级标题

二级标题

段落

使用一个空行来进行分隔多行文本即可形成段落，段落不需使用空格或者制表符记性缩进

段落1

段落2

换行

在行末尾增加2个或2个以上字符然后回车

first line.
second line.

强调

加粗

可以在单词或词组前后加上2个*号或者下划线，如果在单词中间加粗需要前后加2个 * 号且不能有空格

例如：

练习 **markdown**.
练习 __markdown__.
练习 mark**down**.

效果：

练习 markdown.
练习 markdown.
练习 markdown.

斜体

可以在单词或词组前后加上1个*号或者下划线，如果在单词中间有斜体需要前后加1个 * 号且不能有空格

例如：

练习 *markdown*
练习 _markdown_
练习 mark*down*

效果：

练习 markdown
练习 markdown
练习 markdown

加粗斜体

即将加粗和斜体一起组合使用，所以共有4中组合

例如：

练习 ***markdown***
练习 **_markdown_**
练习 ___markdown___
练习 __*markdown*__

效果：

练习 markdown
练习 markdown
练习 markdown
练习 markdown

段落引用

在一个段落前加上大于号（>）

例如：

>这是第一个段落

效果：

这是第一个段落

引用多个段落可以在段落之间的空行也加上大于号(>)

例如：

>这是第一段落
>
>这是第二段落

效果：

这是第一段落

这是第二段落

引用嵌套

可以在要嵌套的段落前面增加两个大于号>>

例如：

>这是第一个段落
>>这是嵌套段落

效果：

这是第一个段落

这是嵌套段落

引用中可以包含其他 markdown 元素，但不是所有元素都支持，需要自己尝试

例如：
> ###第一个*段落*

效果：

第一个段落

列表

有序列表

在每一行前面增加数字和小数点且和文本之间需有一个空格，第一行必须是1. , 后面每一行数字可以不是递增

例如：

1. 第一行
2. 第二行
3. 第三行

效果：

  1. 第一行
  2. 第二行
  3. 第三行

不按顺序

例如：

1. 第一行
3. 第二行
2. 第三行

效果一样：

  1. 第一行
  2. 第二行
  3. 第三行

缩进嵌套

例如：

1. 第一行
2. 第二行
    1. 嵌套第一行
    2. 嵌套第二行
4. 第三行

效果：

  1. 第一行
  2. 第二行
    1. 嵌套第一行
    2. 嵌套第二行
  3. 第三行

无序列表

使用中划线（-）、星号（*）、加号（+）在每一项前面且加上一个空格，嵌套使用缩进即可

例如：

- 第一项
* 第二项
+ 第三项

效果：

  • 第一项
  • 第二项
  • 第三项

嵌套

例如：
- 第一项
* 第二项
    + 嵌套第一项
    + 嵌套第二项
+ 第三项

效果：

  • 第一项
  • 第二项
    • 嵌套第一项
    • 嵌套第二项
  • 第三项

列表中添加元素

要在列表中添加其它元素, 需要缩进4个空格或1个Tab。

添加段落

例如：

+ 第一项
    缩进段落
+ 第二项

效果：

  • 第一项

    缩进段落

  • 第二项

添加引用

例如：

+ 第一项
    > 第一个引用
+ 第二项

效果：

  • 第一项

    第一个引用

  • 第二项

添加代码块

正常来说代码块需要使用4个空格或一个Tab进行缩进，如果在列表中使用代码块需要使用8个空格或2个Tab(一个Tab占4个空格)

例如：
+ 第一项

<html>
  <head>
    <title>Test</title>
  </head>
</html>

+ 第二项

效果：

  • 第一项

    <html>
  <head>
    <title>Test</title>
  </head>
</html>

  • 第二项

添加图片

添加两个空格即可

例如：
+ 第一项
+ 第二项

    ![Test Image](https://d33wubrfki0l68.cloudfront.net/e7ed9fe4bafe46e275c807d63591f85f9ab246ba/e2d28/assets/images/tux.png)

+ 第三项

这里 ![]() 写法是引用图片

效果：

  • 第一项

  • 第二项

  • 第三项

代码

可以使用反向单引号（`）将单词或词组包含起来

例如：

At the command prompt, type `nano`. `code code code `

效果：

At the command prompt, type nano. code code code

转义code

当代码中存在反向引号时可以使用反向双引号实现 code效果

例如：

At the command prompt, type `` `nano` ``.

效果：

At the command prompt, type `nano`.

代码块

将代码所在的每一行都缩进至少4个空格或1个Tab

<html>
  <head>
    <title>Test</title>
  </head>
</html>

水平线

在一行中使用至少3个星号（***）、中划线（—）、下划线（___\）

例如：

***
-–
___

效果：

链接

将文本添加链接，可以使用中括号将文本括起来，然后紧跟圆括号包含url

例如：

测试 [百度](http://www.baidu.com)

效果：

测试百度

给链接增加title

在圆括号的链接后增加空格后再添加即可

例如：

测试[百度](http://www.baidu.com “这是百度链接”)

效果：

测试 百度

将url和邮件文本变成可点击链接

一般来说markdown处理器会直接识别，但也可使用尖括号将其括起来

例如：

\http://www.baidu.com
\test@email.com

效果：

http://www.baidu.com
test@email.com

格式化链接

可以使用 ** 强调链接

例如：

经常使用 **[Google](http://www.google.com)**

效果：

经常使用 Google

引用链接

当觉得文本中的链接太长，可以使用引用链接进行简化。引用链接包含两部分：

  • 正文中使用格式为 [Text][label] ，中间空格可有可无
    第一个中括号的内容为显示的文本，第二个中括号内唯一的标签
  • 文档的其他任何地方使用 [label]: url title
    中括号的内容和正文中的 label 一致，紧跟一个冒号， 冒号之后至少一个空格， 后面就是 url 和 title，title 可以使用单引号、双引号、圆括号包含起来

例如：

这是一段测试文本，可以测试引用 [Google][1]

[1]: http://www.google.com “google”

效果：

这是一段测试文本，可以测试引用 Google

图片

和使用链接的格式类似，在中括号前面加上感叹号，格式为 ![alt](imagePath title) alt 为不能显示图片时的替代文本，imagePath
可以为图片路径或 url，后面的 title 可选

例如：

![图片显示失败](https://d33wubrfki0l68.cloudfront.net/eab45e25bb79970178fab7a2d10cba0209372a59/94d9e/assets/images/philly-magic-garden.jpg “涂鸦”)

效果：

图片链接

即将图片和链接结合起来使用

例如：

[![图片显示失败](https://d33wubrfki0l68.cloudfront.net/eab45e25bb79970178fab7a2d10cba0209372a59/94d9e/assets/images/philly-magic-garden.jpg “涂鸦”)](
http://www.baidu.com)

效果：

转义字符

可以使用反斜线（\）转义 markdown 的字符
可以转义的字符有 \, `, *, -, {}, [], (), #, +, -, ., !

例如：

\* 这是第一项

效果：

* 这是第一项
