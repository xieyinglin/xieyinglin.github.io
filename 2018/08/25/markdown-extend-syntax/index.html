Markdown 扩展语法

发表于 2018-08-25 | 分类于 Markdown |

在基础的语法上扩展可以实现表格，代码块，语法高亮，URL自动链接，脚注功能。这些可以被轻量的标记语言或兼容的 markdown 处理器来实现。

表格

使用至少3个中划线来创建列标题，单元格之间使用 | 进行分隔, 单元格结尾的 | 可选。
例如:

|Syntax     | Description |
|——— | ———– |
|Header    | Title      |
|Paragh    | Text      |
|Line         | Line text |

效果如下：

Syntax Description
Header Title
Paragh Text
Line Line text

表格对齐

表格中的列默认左对齐，可以通过在标题行的中划线左侧，两侧，右侧分别加冒号（:）代表向左对齐，居中，向右对齐

例如：

|Syntax | Description | Test Text |
|:——– | ———–:| ———: |
|Header | Title | first |
|Paragh | Text | second |
|Line | Line text | third |

效果如下：

Syntax Description Test Text
Header Title first
Paragh Text second
Line Line text third

格式化表格中的文本

可以对表格中的文本使用链接，code(使用 ` ,不是代码块)，加粗。
不能使用标题，块引用，列表，水平线，图片，html 标签。

转义表格中的管道符号

可以使用 html 字符码（& #124;）来进行转义管道符（|）

包围的代码块

正常的代码块需要每行都缩进4个空格或1个Tab，如果编辑器支持可以在代码块前后增加一行添加三个` 或者三个 ~

例如：

```
{
“firstName”: “John”,
“lastName”: “smith”,
“age”: 25
}
```

markdown 支持对包围的代码块进行语法高亮，可以在开始行的标记后指明代码块的语言。

例如：

脚注

脚注可以对文本中的某个词组进行解释，语法是：[^标识符]， 标识符可以是数字或文本（不能包含空格或Tab），然后在文档其他除了部分元素（）内添加脚注说明。语法是 [^标识符]: 说明。

例如：

Here’s a simple footnote,[^1] and here’s a longer one.[^bignote]

[^1]: This is the first footnote.

[^bignote]: Here’s one with multiple paragraphs and code.

效果：

Here’s a simple footnote,[^1] and here’s a longer one.[^bignote]

[^1]: This is the first footnote.

[^bignote]: Here’s one with multiple paragraphs and code.

如果显示不正常，则说明该程序不支持

删除线

在文本前后增加两个波浪符号即可实现。

例如：

~~The world is flat.~~ We now know that the world is round.

效果：

The world is flat. We now know that the world is round.

任务列表

任务列表就是开始带有多选框的列表。
语法：- [x] , 在列表项的开始使用 - ,然后一个空格后使用中括号，里面的x代表选中

例如：

- [x] Write the press release
- [ ] Update the website
- [ ] Contact the media

效果如下：

  • Write the press release
  • Update the website
  • Contact the media

自动链接

正常来说，我们在 markdown 文件中添加url会自动识别为可点击链接，如果需要禁用该功能，可以在链接使用code即 `.

例如：

http://www.baidu.com

添加code之后：

`http://www.baidu.com`

效果：

http://www.baidu.com
